## Running the Audit

```bash
npm run audit:deployment
```

This will:
1. Load the deployment milestone configuration
2. Verify all three audit categories
3. Display the Michelin vs. Drive-Thru scorecard
4. Output detailed results with pass/fail status
5. Exit with code 0 if all checks pass, 1 if any fail

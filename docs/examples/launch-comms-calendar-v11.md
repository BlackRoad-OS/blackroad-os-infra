## ğŸ¯ Best Practices

### Pre-Launch (Day -7 to Day 0)
- [ ] Create launch calendar and share with team
- [ ] Assign owners for each channel
- [ ] Draft all message sets (M1-M7)
- [ ] Get approvals from stakeholders
- [ ] Schedule posts/emails in advance
- [ ] Brief support team on incoming feature

### During Launch Week (Day 1-7)
- [ ] Update calendar daily with status
- [ ] Monitor each channel for engagement
- [ ] Address questions promptly
- [ ] Adjust messaging based on feedback
- [ ] Coordinate with support on issues

### Post-Launch (Day 8+)
- [ ] Send follow-up survey (M7)
- [ ] Analyze engagement metrics
- [ ] Document lessons learned
- [ ] Archive launch materials
- [ ] Update launch playbook

## 📝 Usage Instructions

This is a template for tracking a program-of-programs dashboard. To use:

1. **Fill in the dashboard header:**
   - Replace `____________________` with your program name
   - Replace `____________` with your time window (e.g., "Q1 2026", "Jan-Mar")
   - Update the status indicator (🟢/🟡/🔴)

2. **Name your programs:**
   - Replace each `Prog __` with actual program names (e.g., "Prog Alpha", "Prog Beta")

3. **Update status bars:**
   - Each portfolio/program combination shows progress with 7 circles
   - 🟢 = milestone/phase completed successfully
   - 🟡 = milestone/phase at risk or needs attention
   - 🔴 = milestone/phase blocked or failed
   - ⚪️ = milestone/phase not started or not applicable

4. **Monitor portfolio summaries:**
   - The top section shows aggregated status for each portfolio
   - Use emoticons (😭, 🎉, etc.) to highlight critical issues or wins

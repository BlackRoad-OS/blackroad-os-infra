## Overview

Reflection exercises transform individual experiences into team wisdom and organizational knowledge. This guide provides **2 comprehensive reflection systems**:

1. **Team Retrospective Facilitator** - Weekly/monthly team reflection
2. **Learning & Knowledge Synthesis** - Individual and collective learning capture

Together, these systems create a culture of continuous learning and improvement.

#### 3. Facilitation Guide

**Complete 6-phase meeting structure:**

**Phase 1: Check-In (5 min)**
- Emotional temperature check
- Build connection
- Set tone

**Phase 2: Gather Data (15 min)**
- Silent brainstorming
- Post cards to board
- Collect perspectives

**Phase 3: Generate Insights (20 min)**
- Group similar cards
- Discuss themes
- Vote on priorities
- Ask powerful questions

**Phase 4: Decide Actions (15 min)**
- START doing
- STOP doing
- CONTINUE doing
- TRY as experiment
- SMART action items

**Phase 5: Appreciation (5 min)**
- Express gratitude
- Recognize invisible work
- End positively

**Phase 6: Close (5 min)**
- Retro the retro
- Confirm action items
- Schedule next session

---

### Retrospective Formats

**Weekly Sprint Retro (45 min)**
- Focus: Last week's work
- Frequency: Every Friday
- Questions: What went well, what didn't, actions

**Monthly Review (60 min)**
- Focus: Longer-term patterns
- Include: Metrics review, skill development
- Deeper: Strategic discussions

**Quarterly Deep Dive (90 min)**
- Focus: Big picture, team health
- Include: Individual reflections, team dynamics
- Activities: Futurespective, health check

**Project Postmortem (60-90 min)**
- Focus: Completed project
- Include: Timeline review, lessons learned
- Document: For future projects

**Incident Review (45 min)**
- Focus: Specific incident
- Include: Timeline, root cause, preventions
- Blameless: Focus on systems

**Success Celebration (30 min)**
- Focus: Wins, achievements
- Include: Recognition, bonding
- Energy: Positive, energizing

---

### Facilitation Principles

**Create Safety**
- Vegas rule - what's said stays here
- No blame, focus on systems
- Everyone's voice matters
- Respect communication styles

**Stay Neutral**
- Don't advocate solutions
- Help team discover insights
- Manage time, not content
- Encourage diverse perspectives

**Generate Energy**
- Keep it moving
- Use visual techniques
- Mix activities
- End on positives

**Drive Action**
- Insights need action
- Clear owners and deadlines
- Follow up ruthlessly
- Celebrate completion

## Implementation Guide

### Getting Started

**Week 1: Try a Retrospective**
1. Schedule 45-minute retro for Friday
2. Share retrospective questions in advance
3. Use facilitation guide
4. Document action items
5. Follow up

**Week 2: Start Learning Capture**
1. Fill out weekly learning template
2. Share with team
3. Discuss in standup or async
4. Identify one action

**Week 3: Add Daily TIL**
1. Write first TIL
2. Share in team channel
3. Build habit (3 days this week)
4. Review at week end

**Week 4: Synthesize Knowledge**
1. Review all learnings
2. Identify patterns

## Best Practices

### For Retrospectives

**Preparation:**
- Share materials in advance
- Review metrics beforehand
- Identify discussion topics
- Set up space/tools

**During:**
- Start and end on time
- Everyone participates
- Focus on systems not people
- Convert insights to actions

**After:**
- Share summary within 24 hours
- Create issues for action items
- Track progress in standups
- Review before next retro

---

### For Learning Capture

**Make It Easy:**
- Templates readily available
- 5-minute time box
- Focus on specific examples
- Don't aim for perfection

**Make It Regular:**
- Set calendar reminders
- Part of shutdown routine
- Consistent schedule
- Build the habit

**Make It Social:**
- Share with team
- Discuss in standups
- Celebrate learnings
- Learn from each other

**Make It Actionable:**

## Troubleshooting

**Problem:** No time for reflection

**Solution:**
- Start with 5-min TIL only
- Calendar block for reflection
- Make it part of existing meetings
- Focus on highest-value reflections

---

**Problem:** People don't participate

**Solution:**
- Lead by example - share yours first
- Make it psychologically safe
- Celebrate all contributions
- Start with appreciations

---

**Problem:** Same issues every retro

**Solution:**
- Escalate systemic issues
- Try different retro format
- Deep-dive root cause analysis
- Get external help if needed

---

**Problem:** Learnings not applied

**Solution:**
- Every learning needs one action
- Assign owners and deadlines
- Review in standups
- Celebrate when applied

---

**Problem:** Too much to write

**Solution:**
- Use templates
- Focus on top 1-3 learnings
- Bullet points OK
- Voice notes then transcribe
